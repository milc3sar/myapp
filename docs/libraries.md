# Bibliotecas Recomendadas

Este documento presenta las bibliotecas seleccionadas para el desarrollo de la aplicación de Supervisión de Obras.

## Gestión de Estado
**flutter_bloc** - Implementación del patrón BLoC para gestión de estado en toda la aplicación.

## Almacenamiento Local
**hive** - Base de datos NoSQL ligera y rápida para almacenamiento de reportes, evidencias y configuraciones.

## Manejo de Imágenes
**image_picker** - Plugin para seleccionar imágenes de la galería o cámara para la captura de fotos para evidencias.

## Generación de PDF
**pdf** - Biblioteca para crear documentos PDF para la generación de informes en formato PDF.

## Reconocimiento de Voz
**speech_to_text** - Plugin para convertir voz a texto para la transcripción de observaciones.

## Compartir Archivos
**share_plus** - Plugin para compartir contenido para compartir informes PDF.

## Manejo de Permisos
**permission_handler** - Plugin para solicitar permisos para la gestión de permisos de cámara, micrófono y almacenamiento.

## Validación de Formularios
**formz** - Biblioteca para validación de formularios para la validación de datos de entrada en formularios.

## Generación de Código y Modelos Inmutables
**freezed** - Biblioteca para generación de código que reduce el boilerplate en clases de datos inmutables, proporcionando implementaciones automáticas para copyWith, toString, igualdad y serialización JSON.

**json_serializable** - Biblioteca para generación automática de código para convertir clases Dart a y desde JSON.

## Geolocalización y Mapas
**geolocator** - Plugin para acceder a la ubicación del dispositivo para la captura automática de la ubicación donde se toman las fotos.

**flutter_map** - Widget de mapa para Flutter basado en leaflet.js para la generación de imágenes de mapa para mostrar la ubicación en el informe PDF.
